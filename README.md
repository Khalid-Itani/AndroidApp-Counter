# AndroidApp-Counter

A simple **tap counter Android app** built using **Kotlin** and **ConstraintLayout** as part of **CodePath Unit 1 Lab**.

The app is themed around **space**, featuring a galaxy background and a purple planet-style button that users tap to increase their score.

  - After upgrading, each tap increments the counter by **2**
